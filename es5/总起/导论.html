“脚本语言”（script language） 嵌入式（embedded）语言 只能用来做一些数学和逻辑运算，本身不提供任何与 I/O（输入/输出）相关的 API，都要靠宿主环境（host）提供 浏览器，服务器环境，Node 项目。 基本的语法构造（比如操作符、控制结构、语句）和标准库（就是一系列具有各种功能的对象比如Array、Date、Math等） 浏览器控制类：操作浏览器 DOM 类：操作网页的各种元素 Web 类：实现互联网的各种功能 基本语法 标准库 浏览器 API DOM 操纵浏览器的能力 广泛的使用领域 浏览器的平台化——HTML5 Node——服务器，嵌入式 数据库操作：NoSQL, PostgreSQL 移动平台开发：PhoneGap, React, 内嵌脚本语言：Acrobat, Linux GNOME3 跨平台桌面应用程序：Chrome App, Electron, TideSDK 性能： 灵活的语法，表达能力强： 支持过程式/函数式编程， 处理并发（concurrent） 适合异步， 所有值都是对象，方便创造数据结构，不用预定义 支持编译运行 解释性语言编译运行，效率高 WebAssembly 事件驱动和非阻塞设计 适合高并发
